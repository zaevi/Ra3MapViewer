# Ra3MapViewer

红色警戒3地图浏览器

这个工具可以自动扫描地图文件夹，并提供地图预览（包括玩家出生点）。

点击下方操作按钮扫描文件夹，对着地图右键可以导入或删除，你可以：

1. 扫描外部目录，导入指定地图到游戏地图文件夹。
2. 将地图文件夹移动到回收站。

主要是方便导入地图包，移除掉不想要的地图。

目前地图预览的显示略有瑕疵，不影响使用。老系统如果无法打开请安装`.Net Framework 4.7.2`
